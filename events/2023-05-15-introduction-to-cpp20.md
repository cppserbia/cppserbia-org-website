# Introduction to C++20 Features

Join us for an exciting evening exploring the new features in C++20! This session will be led by our community expert, Marko Popović.

## What You'll Learn

- **Concepts**: Learn how to use concepts to constrain templates and make error messages more readable
- **Ranges**: Discover the new ranges library and how it can simplify your code
- **Coroutines**: Understand how coroutines work and how they can make asynchronous code more readable
- **Modules**: Get an introduction to the new module system and how it can replace header files

## Schedule

- 6:00 PM - 6:15 PM: Welcome and networking
- 6:15 PM - 7:15 PM: Main presentation
- 7:15 PM - 7:45 PM: Q&A session
- 7:45 PM - 8:00 PM: Wrap-up and future events

## Prerequisites

- Basic knowledge of C++
- Laptop with a C++20 compatible compiler if you want to follow along

## About the Speaker

Marko Popović is a senior C++ developer with over 10 years of experience in the industry. He has been working with C++20 features since their early drafts and is passionate about sharing his knowledge with the community.

We look forward to seeing you there!

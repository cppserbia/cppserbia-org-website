# Drugo okupljanje C++ user grupe

Drugo okupljanje C++ user grupe održaće se 26. januara, sa početkom u 19h, u beogradskom Startit Centru.

Diskusiju će voditi Goran Aranđelović, softver developer u Endavi.

Tema ovog okupljanja je C++17 standard i nove stvari u jeziku i standardnoj biblioteci. Kako se nekada zbog prirode posla ili projekta na kome radimo nove stvari sporije usvajaju, fokus će biti na primerima koji se tiču svakoga, a ne isključivo library developer-a.

Google grupa: http://groups.google.com/forum/#!forum/std-serbia

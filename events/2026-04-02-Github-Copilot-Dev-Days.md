# GitHub Copilot Dev Days | Belgrade

Join us for a community-led developer event focused on AI-assisted coding with GitHub Copilot. Come and see how to leverage GitHub Copilot on a C++ codebase. Experience this guided tour hands-on with other developers.

Whether you are new to GitHub Copilot or already using it, this event will help you better understand how to apply AI-assisted coding techniques in everyday development.
